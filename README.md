# Bank-Loan-Analysis-
This project focuses on analyzing bank loan data to understand loan distribution, customer behavior, repayment patterns, and risk factors. An interactive Power BI dashboard was developed to provide clear insights for business and decision-making purposes.
Objectives:
1)Analyze total loan amount and payment trends
2)Compare verified vs non-verified loan performance
3)Study grade-wise revolving balance and loan quality
4)Identify state-wise and year-wise loan distribution patterns
Key Dashboard Insights:
1)Verified loans contribute higher total payments compared to non-verified loans
2)Loan grades A and B hold the highest revolving balances
3)Certain states show higher loan concentration, identified using map visualization
4)Year-wise trends reveal changes in loan issuance over time
Tools & Technologies Used:
1)Power BI
2)DAX (Measures & Calculations)
3)Excel (Data Cleaning & Preparation)
4)Data Visualization & Dashboard Design
Dashboard Features :
1)Interactive slicers and filters
2)Bar charts, stacked visuals, and KPI cards
3)State-wise map visualization
4)Clean and user-friendly dashboard layout
Repository Contents:
1)Power BI .pbix dashboard file

4)Project documentation
